## Техническое задание: Рефакторинг и расширение

**Внимание:** Все изменения должны вноситься только в ветку `develop`. Прямые коммиты в `main` запрещены правилами проекта.

### Клонирование проекта

1. Сделай Fork этого репозитория
1. Клонируй свой форк на свой рабочий компьютер

### Доработка проекта

1. Настройка личности (Git Config):
    Перед началом работы убедитесь, что ваши коммиты подписаны вашим реальным именем. Выполните в терминале:
    
    ```bash
    git config user.name "Имя Фамилия"
    git config user.email "ivanov@example.com"
    ```
    _(Это обязательное условие, автопроверка проверяет имя автора последнего коммита)_
1. В первой строке файла calc.py добавьте комментарий со своими Именем и Фамилией \
    Например: `# Автор: Иван Иванов`
1. Создайте ветку `develop` от ветки `main` и переключитесь на неё.
1. В файле `calc.py` исправьте функцию `multiply`, чтобы она возвращала произведение.
1. Создайте новую функцию `sqrt(x)`, которая возвращает квадратный корень числа (используйте `math.sqrt`).
1. Сделайте коммит с информативным сообщением, например: `fix bugs and add sqrt` 
1. Отправьте ветку develop на GitHub
